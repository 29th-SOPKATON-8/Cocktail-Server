## Code Convention / 코드 컨벤션

- Feat: 새로운 기능 추가
- Fix: 버그를 고친 경우
- Style: 코드 포맷 변경
- Refactor: 코드 리팩토링
- Merge: 브랜치 머지
- Docs: 문서 작성
- Rename: 파일, 클래스, 메서드명, 폴더명 수정
- Chore: 설정 파일 등
